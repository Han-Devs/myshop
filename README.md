# 🛒 MyShop

MyShop is a modern responsive e-commerce website built with React and Vite.  
It includes customer shopping features, authentication, product management, admin dashboard, dark mode, and a polished responsive UI.

## 🌍 Live Demo

https://myshop-alpha-inky.vercel.app

## ✨ Features

- Product search
- Category filtering
- Product sorting
- Product details page
- Related products
- Wishlist
- Cart with quantity controls
- Checkout
- Order history
- Customer register/login/logout
- Customer profile
- Username change
- Password change with current password verification
- Protected customer routes
- Admin login/logout
- Admin dashboard
- Add, edit, and delete products
- Image upload from PC
- Stock management
- Featured product toggle
- Sales chart
- Light/Dark mode
- Responsive desktop, tablet, and mobile design
- Mobile hamburger menu
- Scroll to top on route change
